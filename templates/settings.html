How to get your Cloudflare API Token

  1. Go to Cloudflare API Tokens
  2. Click "Create Token"
  3. Use the "Custom token" template
  4. Set the following permissions:
    • Zone:DNS:Edit - Required for DNS challenge
    • Zone:Zone:Read - Required to list zones
  5. Set "Zone Resources" to include the zones you want to manage
  6. Click "Continue to summary" and then "Create Token"
  7. Copy the generated token and paste it above
